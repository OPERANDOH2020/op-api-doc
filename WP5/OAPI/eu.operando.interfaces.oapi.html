BigData

Up 
get /bda/reports
(bdaReportsGet)
provides a link to download a report

Request headers

Query parameters

jobId (required)
Query Parameter — Identification of the job to get the status about

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successful response

400

The request was invalid, e.g. an invalid document format.

401

The user is not authenticated with the OPERANDO system. Check that the service ticket provided by the authentication service is correctly included in the message body.

404

The report with the requested report id could not be found

Reports

Up 
get /reports/{report_id}
Get a report. (reportsReportIdGet)
Called by a regulator to obtain a report matching the given ID. Intended (at the moment) to return a compliance report relating to an OSP.

Path parameters

report_id (required)
Path Parameter — the unique identifier of a report.

Request headers

Query parameters

format (required)
Query Parameter — the requested format of the report (e.g. pdf, html).

Return type

String

Example data

Content-Type: application/json
"aeiou"

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

The request was successful. The report is returned in the response body in the requested format. String

400

The request was invalid, e.g. an invalid document format.

401

The user is not authenticated with the OPERANDO system. Check that the service ticket provided by the authentication service is correctly included in the message body.

404

The report with the requested report id could not be found

PrivacyPolicy

  • policy_description : String
    The description of the privacy policy e.g Doctor access to medical record
  • policy_category : String
    The sector the policy applies to e.g. healthcare
  • recipient_role : String
    The role(s) of persons involved in the policy that receives the private data e.g. doctor, admin
  • action : String
    The action performed on the data e.g. read, write, store, publish, etc.
  • policy_data : String
    The reference to the data object the policy applies to.
  • policy_purpose : String
    The purpose of the this action on the stated private data.

This is the API

privacyPolicyOspIdSettingsPost

Called by the management console when a new B2C OSP registers with OPERANDO. The OSP provides to OPERANDO its privacy settings. These are stored and used in later computations.

URL

http://operando.eu//PrivacyPolicy/{osp_id}/settings/

HTTP Method

POST

Response Type

Parameters

  • Path: osp_id BigDecimal

    The unqiue OPERANDO ID for the registered OSP service.

  • Body: osp_settings List

    The OSPs set of privacy settings.

privacySettingsUserIdOspIdGet

Simple retrieval of an OPERANDO registered user's OSP privacy settings.\nThis method is called by the watchdog component when it requests the settings last applied by the OSE component.\n\nPre condition -- An OPERANDO user must have registered with the OPERANDO platform and subscribed to the OSP service in question.\n\nPre condition --The user's UPP must be stored in the Policy DB component and contain the privacy settings for the OSP service in question.

URL

http://operando.eu//PrivacySettings/{user_id}/{osp_id}/

HTTP Method

GET

Response Type

List

Parameters

  • Path: user_id BigDecimal

    The unqiue OPERANDO ID for the individual user. This corresponds to the unqiue ID of the user record stored in the User Account DB. This corresponds to the identifier of the UPP of the user stored in the Policy DB.

  • Path: osp_id BigDecimal

    The unqiue OPERANDO ID for a given OSP service. Identifies an OSP within OPERANDO.

This is the API

privacyPolicyRegIdPut

Called by the Regulator API. A change to a regulation is entered into OPERANDO. Existing OSPs are then evaluated to see if they comply with the regulation. If not they are sent a report about how to comply.\n\nPre-condition -- The regulation must have been written to the system.

URL

http://operando.eu//PrivacyPolicy/{reg_id}/

HTTP Method

PUT

Response Type

Parameters

  • Path: reg_id BigDecimal

    The unqiue regulation ID for the changed regulation.

  • Body: regulation List

    The description of the changed regulation.

privacyPolicyRegIdPost

Called by the Regulator API. A New regulation is entered into OPERANDO. Existing OSPs are then evaluated to see if they comply with the regulation. If not they are sent a report about how to comply.

URL

http://operando.eu//PrivacyPolicy/{reg_id}/

HTTP Method

POST

Response Type

Parameters

  • Path: reg_id BigDecimal

    The unqiue regulation ID for the new regulation created.

  • Body: regulation List

    The description of the new regulation.

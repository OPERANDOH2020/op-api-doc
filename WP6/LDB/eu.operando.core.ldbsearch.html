LogSearch

Up 
get /operando/core/ldbsearch/log/search
Search logs in database (getLogs)
Search logs in database by specifying a filter.

Query parameters

dateFrom (optional)
Query Parameter — Date from which wanted to be recovered the logs.
dateTo (optional)
Query Parameter — Date to which wanted to be recovered the logs.
logLevel (optional)
Query Parameter — Log level wanted to be recovered.
requesterType (optional)
Query Parameter — Type of the requester that originated the log entry.
requesterId (optional)
Query Parameter — Id of the requester that originated the log entry.
logPriority (optional)
Query Parameter — Priority of the log messages to be recovered.
title (optional)
Query Parameter — Title of the log messages to be recovered.
keyWords (optional)
Query Parameter — Keywords to perform the search.
logType (optional)
Query Parameter — Type of the data logged.
affectedUserId (optional)
Query Parameter — Id of the affected user.

Return type

inline_response_200

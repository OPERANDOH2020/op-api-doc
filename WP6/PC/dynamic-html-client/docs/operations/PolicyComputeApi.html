This is the API

computeUPPPost

Called by the management console when a user subscribes to a new OSP.

URL

http://operando.eu/policycomputation/computeUPP

HTTP Method

POST

Response Type

Report

Parameters

  • Query: osp_id String

    OSP unique identifier of the OSP to compute and add to the UPP

  • Query: user_id String

    user unique identifier of the UPP to compute

  • Body: osp_request List(OSPDataRequest)

    The set of requests that the OSP will carry out on a user's private data. This will likely be stored in the DB, so not needed to pass each time.

computeUPPPreferencesPost

Called by the management console when a user enters their information into questionnaire.

URL

http://operando.eu/policycomputation/computeUPPPreferences

HTTP Method

POST

Response Type

Report

Parameters

  • Query: user_id String

    user unique identifier

  • Body: user_preferences List(UserPreference)

    The array of preference values.

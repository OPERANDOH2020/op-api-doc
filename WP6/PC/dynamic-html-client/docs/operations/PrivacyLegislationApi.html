This is the API

regulationsRegIdPost

Called by the Regulator API. A New regulation is entered into OPERANDO. The regulation is stored in the policy DB. Existing UPPs are then evaluated to see if they comply with the regulation.

URL

http://operando.eu/policycomputation/Regulations/{reg_id}/

HTTP Method

POST

Response Type

Parameters

  • Path: reg_id BigDecimal

    The unqiue regulation ID for the new regulation created.

  • Body: regulation List(PrivacyRegulation)

    The description of the new regulation.

regulationsRegIdPut

Called by the Regulator API. An updated regulation is entered into OPERANDO. The regulation is stored in the policy DB. Existing UPPs are then evaluated to see if they comply with the regulation.\n\n\nPre-condition -- The regulation must have been written to the system.

URL

http://operando.eu/policycomputation/Regulations/{reg_id}/

HTTP Method

PUT

Response Type

Parameters

  • Path: reg_id BigDecimal

    The unqiue regulation ID for the changed regulation.

  • Body: regulation List(PrivacyRegulation)

    The description of the changed regulation.

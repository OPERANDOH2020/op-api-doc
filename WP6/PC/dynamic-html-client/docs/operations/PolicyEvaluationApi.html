This is the API

evaluateOSPRequestPost

Called by the Rights Management Component when evaluating if an OSP request or set of requests matches the user preferences stored in the user's UPP.

URL

http://operando.eu/policycomputation/evaluateOSPRequest

HTTP Method

POST

Response Type

Report

Parameters

  • Query: user_id UUID

    Unique identifier representing a specific user

  • Query: osp_id String

    OSP unique identifier

  • Body: osp_request List(OSPDataRequest)

    The set of requests that the OSP wants to on a user's private data

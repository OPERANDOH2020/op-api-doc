This is the API

oSPOspIdPost

Called by the policy computation component when a new regulation is added to OPERANDO.

URL

http://operando.eu/policyDB/OSP/{osp_id}/

HTTP Method

POST

Response Type

Parameters

  • Path: osp_id BigDecimal

    Unique identifier representing a specific osp

  • Body: regulation OSPPrivacyPolicy(OSPPrivacyPolicy)

    The first instance of this OSP document

regulationsRegIdPost

Called by the policy computation component when a new regulation is added to OPERANDO.

URL

http://operando.eu/policyDB/Regulations/{reg_id}/

HTTP Method

POST

Response Type

Parameters

  • Path: regulation_id BigDecimal

    Unique identifier representing a specific user

  • Body: regulation PrivacyRegulation(PrivacyRegulation)

    The first instance of this regulation document

userPrivacyPolicyUserIdPost

Called when a new user is registered with operando. Their new privacy preferences are passed in the UPP document and stored in the policy DB.

URL

http://operando.eu/policyDB/UserPrivacyPolicy/{user_id}

HTTP Method

POST

Response Type

Parameters

  • Path: user_id BigDecimal

    Unique identifier representing a specific user

  • Body: upp UserPrivacyPolicy(UserPrivacyPolicy)

    The first instance of this user's UPP

This is the API

oSPOspIdPut

Called when by the policy computation component when the regulator api updates a regulation. Their new OSPRequest document is stored in the policy DB.

URL

http://operando.eu/policyDB/OSP/{osp_id}/

HTTP Method

PUT

Response Type

Parameters

  • Path: osp_id BigDecimal

    Unique identifier representing a specific osp

  • Body: regulation OSPPrivacyPolicy(OSPPrivacyPolicy)

    The changed instance of this OSPRequest

regulationsRegIdPut

Called when by the policy computation component when the regulator api updates a regulation. Their new PrivacyRegulation document is stored in the policy DB.

URL

http://operando.eu/policyDB/Regulations/{reg_id}/

HTTP Method

PUT

Response Type

Parameters

  • Path: regulation_id BigDecimal

    Unique identifier representing a specific regulation

  • Body: regulation PrivacyRegulation(PrivacyRegulation)

    The changed instance of this PrivacyRegulation

userPrivacyPolicyUserIdPut

Called when a user makes a change to the UPP registered with operando. Their new privacy preferences are passed in the UPP document and stored in the policy DB.

URL

http://operando.eu/policyDB/UserPrivacyPolicy/{user_id}

HTTP Method

PUT

Response Type

Parameters

  • Path: user_id BigDecimal

    Unique identifier representing a specific user

  • Body: upp UserPrivacyPolicy(UserPrivacyPolicy)

    The changed instance of this user's UPP

This is the API

userPrivacyPolicyGet

The query contains a json object (names, values) and the request returns the list of documents (UPPs) where the filter matches i.e. the document contains fields with those values.

URL

http://operando.eu/policyDB/UserPrivacyPolicy/

HTTP Method

GET

Response Type

List

Parameters

  • Query: filter String

    The query filter to be matched - \"?filter=\"{json description}\"

userPrivacyPolicyUserIdGet

Get a specific UPP document via the user's id. This will return the full user privacy policy document in json format.

URL

http://operando.eu/policyDB/UserPrivacyPolicy/{user_id}

HTTP Method

GET

Response Type

UPP

Parameters

  • Path: user_id BigDecimal

    Unique identifier representing a specific user

userPrivacyPolicyUserIdPut

Called when a user makes a change to the UPP registered with operando. Their new privacy preferences are passed in the UPP document and stored in the policy DB.

URL

http://operando.eu/policyDB/UserPrivacyPolicy/{user_id}

HTTP Method

PUT

Response Type

Parameters

  • Path: user_id BigDecimal

    Unique identifier representing a specific user

  • Body: upp UPP

    The changed instance of this user's UPP

userPrivacyPolicyUserIdPost

Called when a new user is registered with operando. Their new privacy preferences are passed in the UPP document and stored in the policy DB.

URL

http://operando.eu/policyDB/UserPrivacyPolicy/{user_id}

HTTP Method

POST

Response Type

Parameters

  • Path: user_id BigDecimal

    Unique identifier representing a specific user

  • Body: upp UPP

    The first instance of this user's UPP

userPrivacyPolicyUserIdDelete

Called when a user leaves operando. Their privacy preferences and policies are deleted from the database.

URL

http://operando.eu/policyDB/UserPrivacyPolicy/{user_id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: user_id BigDecimal

    Unique identifier representing a specific user

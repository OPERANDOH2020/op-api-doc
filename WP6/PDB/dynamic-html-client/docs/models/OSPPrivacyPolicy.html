OSPPrivacyPolicy

  • PolicyText : String
    The content of the OSP privacy policy, textually described and published. It is\na the full text adverstised by the OSP.
  • PolicyURL : String
    The url location of the privacy policy of the OSP
  • Workflow : List
    The sequence of requests that this OSP makes (simple ordered array list\nin this version). The requests describes the operations that the OSP may\ncarry out on the data. This can be used for compliance checking and computation\nof user policies.
  • Policies : List
    The list of rights that the OSP intends to follow e.g. give X access to Y data for Z purpose. This\ninformation can then be used in calculation of the policy and in compliance checking.
